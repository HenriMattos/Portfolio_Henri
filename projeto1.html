Sobre:

O CryptCronos é uma ferramenta de criptografia que suporta dois algoritmos: AES e RSA. O script permite criptografar e descriptografar arquivos com segurança usando essas técnicas de criptografia.

Funcionalidades:

  • Criptografia e descriptografia de arquivos usando AES (Advanced Encryption Standard)
  • Criptografia e descriptografia de arquivos usando RSA (Rivest-Shamir-Adleman)
  • Geração e uso de chaves AES e RSA
  • Interface de linha de comando com opções interativas e coloridas

Requisitos:

  • Python 3.x
  • Biblioteca pycryptodome para criptografia
  • Biblioteca colorama para estilização do terminal

Explicação do Código:

  • Função gerar_chave_aes(bits): Gera uma chave AES com o comprimento especificado em bits.
  • Função criptografar_arquivo_aes(arquivo_entrada, arquivo_saida, chave): Criptografa um arquivo usando a chave AES fornecida.
  • Função descriptografar_arquivo_aes(arquivo_entrada, arquivo_saida, chave): Descriptografa um arquivo usando a chave AES fornecida.
  • Função gerar_pares_chaves_rsa(): Gera pares de chaves RSA e salva as chaves privada e pública em arquivos.
  • Função criptografar_arquivo_rsa(arquivo_entrada, arquivo_saida, arquivo_chave_publica): Criptografa um arquivo usando a chave pública RSA fornecida.
  • Função descriptografar_arquivo_rsa(arquivo_entrada, arquivo_saida, arquivo_chave_privada): Descriptografa um arquivo usando a chave privada RSA fornecida.
  • Função imprimir_menu(): Exibe o menu principal com opções para criptografia e descriptografia.
  • Função mostrar_info(opcao): Fornece informações detalhadas sobre cada opção do menu.
  • Função main(): Função principal que executa o menu e gerencia a execução das opções selecionadas.

Ainda preciso implementar:

  • Suporte para criptografia de arquivos grandes em múltiplos blocos
  • Melhor tratamento de erros para todas as operações de criptografia
  • Adição de um sistema de autenticação para maior segurança

Utilização:

  • Salve o código em um arquivo Python (.py)
  • Execute o script com o comando python cryptcronos.py
  • Escolha uma opção no menu para criptografar ou descriptografar arquivos
  • Para criptografia RSA, certifique-se de ter arquivos de chave pública e privada

Observações:

  • Este projeto é para fins educacionais e de demonstração.
  • Utilize as ferramentas com responsabilidade e segurança.
  • Certifique-se de proteger suas chaves de criptografia e não compartilhar informações sensíveis.
