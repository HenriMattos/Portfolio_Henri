EyePortScan - Scanner de Portas

Sobre o Projeto

Este projeto implementa um scanner de portas TCP multithread.

O uso de múltiplas threads torna o escaneamento de portas muito mais rápido.

Funcionalidades

  • Escaneamento multithread de portas TCP.
  • Entrada e validação de um intervalo de portas para o escaneamento.
  • Mostra as portas abertas ou, caso não encontre, informa que não foram detectadas portas abertas.

Requisitos

  • Python 3.x
  • Instalação da biblioteca Rich via pip:
pip install rich

Utilização

  • Execute o script Python com o comando: python portscan.py.
  • Digite o endereço IP ou domínio do alvo.
  • Digite o intervalo de portas que deseja escanear (de 1 a 65535).
  • O script mostrará o progresso do escaneamento e listará as portas abertas no final.
