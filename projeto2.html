Documentação do Projeto

View on GitHub

Sobre:

Este projeto implementa um scanner de portas TCP multithread utilizando a biblioteca Rich para melhorar a interface visual no terminal. O uso de múltiplas threads torna o escaneamento de portas muito mais rápido.

Funcionalidades:

  • Escaneamento multithread de portas TCP.
  • Entrada e validação de um intervalo de portas para o escaneamento.
  • Mostra as portas abertas ou, caso não encontre, informa que não foram detectadas portas abertas.

Requisitos:

  • Python 3.x
  • Instalação da biblioteca Rich via pip:
pip install rich

Instalação:

Basta salvar o código em um arquivo .py e executá-lo após instalar a biblioteca Rich.

Como usar:

  • Execute o script Python com o comando: python portscan.py.
  • Digite o endereço IP ou domínio do alvo.
  • Digite o intervalo de portas que deseja escanear (de 1 a 65535).
  • O script mostrará o progresso do escaneamento e listará as portas abertas no final.

Exemplo de Saída:

        Enter the target IP address or domain: 192.168.1.100
        Enter the start port: 20
        Enter the end port: 100

        Starting scan on 192.168.1.100 from port 20 to 100
        Scanning ports...
        Port 22 is open
        Port 80 is open
        Port 443 is open

        Scan completed. Open ports:
        - 22
        - 80
        - 443

Explicação do Código:

  • Multithreading: Usa múltiplas threads para distribuir a carga de escaneamento entre várias portas simultaneamente, aumentando a velocidade do processo.
  • Biblioteca Rich: A biblioteca Rich é usada para colorir a saída no terminal e adicionar uma barra de progresso que acompanha o andamento do escaneamento.
  • Fila (queue): As portas são adicionadas a uma fila, e cada thread pega uma porta da fila para escanear até que todas as portas sejam verificadas.
  • Validação: O código valida a entrada de endereço IP e o intervalo de portas, garantindo que os valores estejam corretos antes de iniciar o escaneamento.

Possíveis Melhorias:

  • Adicionar suporte para escaneamento UDP.
  • Melhorar a velocidade do escaneamento utilizando técnicas de paralelismo avançado.
  • Implementar uma interface gráfica (GUI) para facilitar o uso por usuários não técnicos.

Observações:

  • Utilize esta ferramenta de forma ética e responsável.
  • Este projeto é destinado apenas para fins educacionais e de pesquisa.
